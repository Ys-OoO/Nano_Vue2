# Nano_Vue2
Nano_Vue2 aims to further understand the inner workings of Vue2

## Implemented Features
* global API:
  * NanoVue.mixin
  * NanoVue.component
  * NanoVue.extend
* options / Data:
  * data
  * watch
  * computed
* options / DOM:
  * el
  * template
  * render
* options / Lifecycle Hook:
  * _pending..._
* instance property:
  * $data
  * $options
  * $el
* instance method / data:
  * $watch
  * $set
* instance method / lifecycle:
  * instance.$mount
  * instance.$nextTick
